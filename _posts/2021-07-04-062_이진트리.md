## 이진 트리

+ 차수가 2 이하인 노드들로 구성된 트리
+ 단말 노드의 개수 = 차수가 2인 노드의 개수 + 1





## 트리의 운행법(Traversal)

+ Preorder 운행

  Root -> Left -> Right

+ Inorder 운행

  Left -> Root -> Right

+ Postorder 운행

  Left -> Right -> Root



## 수식의 표기법

+ PreFix

  이진 트리의 수식을 Preorder

+ InFix

  Inorder

+ PostFix

  Postorder
